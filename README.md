# Initial Setup

Ruby 2.7.1
Rails 6.0.3.2
Node 12.18.3

```
bundle install
rails db:create
rails db:migrate
rails db:fixtures:load
```

# Running Locally

```
foreman start
```

# Tests

## Test Suite

`rails t`

## System Tests

`rails test:system`
